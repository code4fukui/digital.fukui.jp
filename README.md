# digital.fukui.jp

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

[http://digital.fukui.jp/](http://digital.fukui.jp/)

A website that provides digital resources and open data related to Fukui Prefecture, Japan.

## Features

-   Centralizes information on digitalization efforts within Fukui Prefecture.
-   Showcases open data published by the prefecture.
-   Serves as an open-source portal for Fukui-related digital content.

## Usage

The website can be accessed directly at the URL above.

## Data Source

This project utilizes open data published by [Fukui Prefecture](https://www.pref.fukui.lg.jp/).
