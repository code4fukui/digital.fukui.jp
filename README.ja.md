# digital.fukui.jp

[http://digital.fukui.jp/](http://digital.fukui.jp/)

福井県に関連するデジタルリソースとオープンデータを提供するウェブサイトです。

## 特徴

- 福井県内のデジタル化の取り組みに関する情報を一元化します。
- 県が公開するオープンデータを紹介します。
- 福井に関連するデジタルコンテンツのオープンソースポータルとして機能します。

## 使い方

上記のURLから直接ウェブサイトにアクセスできます。

## データソース

本プロジェクトは、[福井県](https://www.pref.fukui.lg.jp/)が公開するオープンデータを利用しています。
